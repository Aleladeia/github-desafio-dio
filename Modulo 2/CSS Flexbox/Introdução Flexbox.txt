Modelo de layout unidimensional

Flex container, tag que envolve os itens, possui itens filhos, a propriedade passada a tag é o "display:flex", uma vez inicializado todos os itens filhos são tranformados em flex itens, essa inicialização de container pode ser feita em qual tag(<div>, <span>, <h1> e etc...)

Propriedades relacionadas.
display = inicializador do container
flex-direction = faz o direcionamento dos itens em linha, coluna...
flex-wrap = aplicar a quebra de linha
flex-flow = abreviação do direction e wrap
justify-content = alinha os itens do container de acordo com sua direção
align-items = alinha os itens de acordo com sei eixo
align-content = alinha as linhas do container.

Flex-item = itens filhos diretos do flex container
